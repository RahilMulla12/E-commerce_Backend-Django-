# Fista Mart

A modern E-commerce web application built using React and Django.

## Features
- User Authentication
- Product Listing
- Product Details
- Shopping Cart
- Checkout System
- Order Management

## Tech Stack
### Frontend
- React
- Bootstrap
- React Router

### Backend
- Django
- Django REST Framework
- JWT Authentication

## Installation

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
pip install -r requirements.txt
python manage.py runserver
```
